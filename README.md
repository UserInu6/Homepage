# Homepage

A fully customizable browser homepage and new tab experience inspired by the best design elements of both Google Chrome and Brave Browser.

Built as a standalone HTML-based dashboard focused on speed, convenience, personalization, and clean aesthetics.

🔗 Live Demo: https://userinu6.github.io/Homepage/

---

# ✨ Overview
This project combines:

- A Chrome-style Google apps launcher
- A modern customizable favorites dashboard
- Dynamic UI states and animations
- Deep visual personalization controls
- Local or online wallpaper support
- A minimal idle mode with expandable workspace functionality

Designed to function as both:
- A browser homepage
- A replacement new tab page

---

# 🚀 Features

## 1. Dynamic Layout States

### Minimal Idle Interface
The homepage launches in a clean, distraction-free state centered around a search bar.

### Expandable Dashboard
Clicking anywhere on the page transitions the interface into a fully interactive dashboard containing shortcut cards and additional controls.

### Smart Search Omnibar
Built-in search engine switching allows instant searches across multiple providers directly from the homepage.

---

## 2. Chrome-Style Google Apps Panel

### Slide-Out Google Workspace Drawer
A pixel-styled 9-dots launcher button opens a dedicated side panel containing quick access to major Google services.

### Fallback Icon Rendering
Custom fallback handling ensures broken or unavailable icons are replaced gracefully with clean placeholders.

---

## 3. Interactive Favorites System

### Custom Shortcut Cards
Add personalized website cards with:
- Custom titles
- Website URLs
- Optional thumbnail images

### Drag-and-Drop Sorting
Freely rearrange cards within the grid layout.

### Context Menu Controls
Right-click functionality enables:
- Editing shortcuts
- Deleting shortcuts

---

## 4. Visual Customization

### Glassmorphism Controls
Real-time sliders allow adjustment of:
- Transparency
- Blur intensity
- Active and idle state opacity

### Layout Matrix Controls
Customize:
- Grid rows
- Grid columns
- Shortcut sizing behavior

### Typography & Styling
Includes:
- Multiple font modes
- Retro pixel-style typography
- Clean modern sans-serif layouts
- Adjustable text scaling

### Wallpaper Customization
Supports:
- Online image URLs
- Local `file:///` image paths
- High-quality custom backgrounds

---

## 5. Additional Enhancements

- Optional daily quotes
- Smooth transitions and animations
- Responsive layout behavior
- Lightweight standalone architecture
- No external framework dependencies

---

# ⚙️ Browser Setup (Brave)

To use as a homepage in Brave Browser:

1. Open `brave://settings/`
2. Navigate to:
   - **Appearance**
   - **Get Started**
3. Set:
   - Startup page
   - New tab override
4. Point to either:
   - Local HTML file
   - GitHub Pages URL
